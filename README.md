# BlogShare
website personal blog with ReactJs and Go-Fiber (on-going)

technology used : 
Backend (Go):
- Fiber (Framework)
- Json Web Token (JWT)
- Authentication
- PostgreSQL
- Bcrypt
- RESTful API

Frontend (ReactJs) :
- Typescript (.TSX)
- Bootstrap & CSS

how to run :

Backend :
```
> cd backend
> go mod tidy
> go run main.go
```

frontend : 
```console
> cd frontend
> npm install
> npm start
